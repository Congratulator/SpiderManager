爬虫管理平台

需求分析

基于对分布式爬虫的管理需要，按顺序整理以下的需求分析。

  1. 查看Scrapyd节点状态；
  2. 查看Scrapyd节点下存在的Project和Spider;
  3. 支持对Spider的分组功能；
  4. 对Spider组和单个Spider进行控制（运行、暂停、继续、停止）；
  5. 实时显示分组及Spiders的工作状态。

概要设计

技术选型
  • 后端框架：Django
  • 数据库：Sqlite3
  • 前端框架：React.js
  • 爬虫部署平台：Scrapyd
